Visualizer2026 WebXR

This repository is the WebXR and immersive VR extension of the Visualizer2026 project. It is designed to explore how linear algebra concepts can be experienced in interactive 3D and VR environments.

Project Status

This WebXR module is currently under development. The public landing page is available now, and the interactive VR experience will be linked here in a future update.

Planned Features

  • Immersive visualization of vectors and matrix transformations
  • Interactive manipulation of 3D coordinate systems
  • WebXR support for VR headsets such as Meta Quest
  • Educational demonstrations connecting geometry and linear algebra
